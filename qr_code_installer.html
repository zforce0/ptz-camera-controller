Installation Instructions:

  1. Scan the QR code with your Android tablet's camera or QR code scanner app
  2. Download the APK file when your browser opens
  3. Enable unknown sources in your tablet's Settings → Security menu
  4. Open the APK file from your Downloads folder to start installation
  5. Follow the prompts to complete installation
  6. Launch the app and follow the setup instructions
Note: To update the APK link:
  1. Build your APK using Android Studio or the automated GitHub Actions workflow
  2. Upload to GitHub Releases or update through the GitHub Actions workflow
  3. The QR code and download link will automatically point to the latest release
